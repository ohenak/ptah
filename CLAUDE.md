# Project: eng-team

## Shell Environment

Node.js 20 LTS is installed via Homebrew. Ensure it's on PATH before running any node/npm/npx commands:

```
export PATH="/opt/homebrew/opt/node@20/bin:$PATH"
```

## Project Structure

- `ptah/` — Ptah CLI (TypeScript, ESM, vitest)

## Common Commands

All commands run from `ptah/` directory:

- Build: `npm run build`
- Test: `npm test`
- Dev: `npm run dev`
